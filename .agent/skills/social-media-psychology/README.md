# 🧠 Social Media Psychology

**Skill de consultoria estratégica** sobre psicologia de redes sociais e algoritmos de distribuição — orienta escrita e valida conteúdo para maximizar engajamento no LinkedIn e Instagram.

## Para que serve?

Esta skill atua como **módulo de conhecimento consultivo** que outras skills (escrita, revisão, design, publicação) consomem para otimizar conteúdo. Ela não gera conteúdo — **orienta a geração e valida na revisão**.

> "O melhor hack de algoritmo é conteúdo que as pessoas realmente querem ler, salvar e compartilhar."

## Modos de Operação

| Modo | Quando | O que retorna |
|------|--------|---------------|
| **Consulta** | Antes/durante a escrita | Diretrizes de Otimização |
| **Revisão** | Após a escrita, antes de publicar | Checklist de Conformidade |

## 4 Pilares de Conhecimento

| Pilar | O que cobre |
|-------|-------------|
| **1. Algoritmos de Distribuição** | Como LinkedIn e Instagram rankeiam conteúdo (sinais, métricas, penalizações) |
| **2. Psicologia de Persuasão** | 6 princípios de Cialdini aplicados à criação de conteúdo |
| **3. Vieses Cognitivos** | Open Loop, Framing, Loss Aversion, Anchoring e outros atalhos mentais |
| **4. Hooks & Anti-Patterns** | Técnicas de abertura que funcionam e padrões que destroem alcance |

## Plataformas Cobertas

| Plataforma | Fatores-Chave |
|------------|---------------|
| **LinkedIn** | Dwell time, comentários significativos, Golden Hour (60-90min) |
| **Instagram Feed** | DM shares (3-5x > likes), saves, originalidade |
| **Instagram Reels** | Watch time, hook nos primeiros 3s, completion rate |
| **Instagram Stories** | Interação, respostas via DM, stickers interativos |

## Integração

A skill é consumida via **composição** por:

- `engineering-writer` → Modo Consulta (diretrizes antes de escrever)
- `engineering-writer-revisor` → Modo Revisão (validação pós-escrita)
- `designer` → Modo Consulta (orientações visuais por plataforma)
- `linkedin-poster` / `instagram-poster` → Modo Revisão (conformidade antes de publicar)

## Quando usar?

- Para otimizar texto para uma plataforma específica
- Para validar se um conteúdo está formatado para engajamento
- Para orientar hooks, CTAs e estrutura de posts
- Para revisar anti-patterns que reduzem alcance
